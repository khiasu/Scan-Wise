# ScanWise

[![Download APK](https://img.shields.io/badge/Download-Latest%20APK-vividpurple?style=for-the-badge&logo=android)](https://github.com/khiasu/Scan-Wise/releases)

ScanWise is a premium local-first Android document scanner and digitizer. It captures physical documents via camera, imports gallery images, or rasterizes PDF files, and extracts structured content directly using secure, developer-configured cloud text extraction engines.

---

## Key Features

- **Download APK First**: Access the latest pre-compiled build directly from [GitHub Releases](https://github.com/khiasu/Scan-Wise/releases).
- **Secure On-Device Storage**: Scan logs, database records, and files are stored inside your device's private storage.
- **Hardware-Backed Encryption**: Credentials (API Keys) are encrypted using AES-256 and backed by the Android Keystore system.
- **Dynamic Themes**: Toggle instantly between Cream Light theme and Purple/Black Dark theme.
- **File Exports**: Export scans to PDF, DOCX, CSV, or JSON straight to your device's public Downloads directory.

---

## Quick Start

1. Download and install the [latest APK](https://github.com/khiasu/Scan-Wise/releases) on your Android device (API 26+).
2. Go to **Settings** and input your Gemini, OpenAI, or Claude developer API key.
3. Tap **Scan** to capture pages and choose **Analyze & Digitize** to begin text extraction.
4. Manage documents and export them from the **Library** tab.
